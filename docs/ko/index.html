옵스뷰 시스템

Windows 원격 데스크톱 및 CCTV 모니터링을 위한 차세대 View-only 스트리밍 아키텍처. 가볍고, 눈부시게 빠르며, 완벽하게 안전합니다.

시스템 접속 깃허브

핵심 기능

델타 스트리밍 엔진

변경 영역(128x128 픽셀)만 zstd로 압축하여 전송합니다. 저사양 PC에서도 완벽한 프레임레이트를 보장합니다.

통합 멀티 뷰어

웹 브라우저나 전용 데스크톱 앱(Wails)을 통해 PC 화면과 CCTV 피드를 동시에 모니터링할 수 있습니다.

제로 트러스트 보안

화면에 표시되는 동적 6자리 PIN 번호로 완벽하게 인증합니다. 내부 LAN 및 외부 Public 웹소켓 통신을 모두 지원합니다.

가변 해상도 프로파일

네트워크 환경에 맞춰 1080p와 720p 해상도 프로파일을 즉시 전환할 수 있습니다.

하이브리드 네트워크

내부망(ws://)과 암호화된 리버스 프록시 외부망(wss://) 모드를 동시에 완벽하게 지원합니다.

CCTV 대시보드

RTSP 프로토콜을 통해 하이크비전 및 다후아 프록시 피드를 직접 연결하여 관제합니다.
